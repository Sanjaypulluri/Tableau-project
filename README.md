# IPL Cricket Analysis Tableau Dashboard

This project presents interactive dashboards built in **Tableau** to analyze performance trends of teams as well as individual players.  

dataset is Indian Premier League (IPL) 2008-2019 dataset. This dataset has all the data for offering comprehensive insights about the performance of teams in the ipl and the individual player performance, like the statistics of the batsman and the bowler along with the player of the matches for the specific match and the umpires for that specific match happened. It also has detailed statistics about the matches and deliveries, which will help in in-depth analysis of the team performance over by over and the graph of score from over 1 to over 20.
